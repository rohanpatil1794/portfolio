## Overview

A specialized QA chatbot built on top of open-source LLMs, grounded with a FAISS vector index built from medical textbooks and research papers.

## Key Features

- FAISS-backed document retrieval over medical literature
- Hugging Face model inference (local + API modes)
- Citation of source passages in responses
- Streamlit interface with chat history
