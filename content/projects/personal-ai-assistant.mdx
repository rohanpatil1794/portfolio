## Overview

A full-stack personal AI assistant that uses Retrieval-Augmented Generation (RAG) to answer questions grounded in your own documents and notes.

## Key Features

- **RAG pipeline** — Ingests PDFs and text files, chunks them, embeds via OpenAI, stores in a vector DB
- **Conversational memory** — Maintains multi-turn conversation history per session
- **FastAPI backend** — Async API with streaming responses
- **React frontend** — Clean chat UI with markdown rendering

## Technical Details

The system uses LangChain's `RetrievalQA` chain with a custom prompt template that keeps responses grounded in retrieved context. Embeddings are stored in ChromaDB for fast similarity search.
